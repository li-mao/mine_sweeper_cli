# 命令行版扫雷
mine_sweeper_cli    
by python3

#### 简介：
```
单文件python小游戏，
可以在服务器上玩的扫雷游戏，
小伙伴们又可以愉快的摸鱼了!!!
```

#### 说明：
```
游戏左侧:'行编号';顶行:'列编号',
根据提示输入'行编号'、'列编号'，就可以打开该格，进行游戏。
有剩余安全格总数，和雷区总数提示。
游戏内 :
    '*' => 未打开格 ,
    [0-8] => 周围雷区总数 ,
    '#' => 雷区 ,
```
